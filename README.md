Было осуществлено программирование микроконтроллера STM32 на базе платы Blue Pill для подсчета частот сигналов с 8 световых датчиков. Восемь результатов передавались с платы на ПК через UART, и после осуществлялось их считывание. Программирование STM32 осуществлялось через библиотеку HAL.
Файл frequency_8_ETR.ioc можно запустить в среде CubeMX, где он собственно был сделан, и наглядно просмотреть назначение выводов платы Blue Pill.
